## ✅ STEP 1: Install Homebrew

Homebrew is a package manager that makes installing software on Mac easy.

### Copy and paste this command in Terminal:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### What will happen:
1. You'll be asked for your Mac password (type it - you won't see it, that's normal)
2. Press **Enter** to continue
3. Wait 1-2 minutes for installation
4. You'll see "Installation successful!"

### After installation, run these TWO commands:

```bash
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
```

Then:

```bash
eval "$(/opt/homebrew/bin/brew shellenv)"
```

### Verify Homebrew is installed:

```bash
brew --version
```

**Expected output:** `Homebrew 4.x.x`

✅ **Once you see the version number, Homebrew is installed! Tell me and we'll move to Step 2.**
